### Napisz program do dostawy pizzy

Gratulacje, dostałeś pracę w Python Pizza! Twoim pierwszym zadaniem jest stworzenie automatycznego programu do zamawiania pizzy.

Na podstawie zamówienia użytkownika oblicz jego ostateczny rachunek. Użyj funkcji `input()`, aby uzyskać preferencje użytkownika, a następnie podsumuj całkowity koszt jego zamówienia i poinformuj, ile musi zapłacić.

Mała pizza (S): 15$

Średnia pizza (M): 20$

Duża pizza (L): 25$

Dodaj pepperoni do małej pizzy (Y lub N): +2$

Dodaj pepperoni do średniej lub dużej pizzy (Y lub N): +3$

Dodaj dodatkowy ser do pizzy dowolnego rozmiaru (Y lub N): +1$

##### Przykład interakcji

```
Witamy w Python Pizza Deliveries!
Jaką rozmiar pizzy chcesz? S, M czy L: L
Czy chcesz pepperoni na swojej pizzy? Y lub N: Y
Czy chcesz dodatkowy ser? Y lub N: N
Twój ostateczny rachunek to: 28$.
```

<div class="hint">
  Nie zmieniaj żadnego z początkowego kodu i upewnij się, że ostatnie zdanie brzmi: "Twój ostateczny rachunek to: $<bill amount>." włącznie z kropką i identycznym sformułowaniem. W przeciwnym razie testy nie przejdą.
</div>
