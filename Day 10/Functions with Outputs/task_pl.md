Widzieliśmy funkcje z samym ciałem wykonawczym, funkcje z argumentami wejściowymi umożliwiające różnorodność w działaniu ciała funkcji, a teraz poznamy ostatnią formę funkcji. Funkcje, które mogą mieć dane wyjściowe.

### PRZERWA 1
Utwórz funkcję o nazwie `format_name()`, która przyjmuje dwa argumenty wejściowe: `f_name` i `l_name`.

### PRZERWA 2
Użyj funkcji `title()`, aby zmodyfikować parametry `f_name` i `l_name` na format z Wielkimi Literami na początku.

### Składnia
Możesz utworzyć funkcję z ciałem, argumentem wejściowym i danymi wyjściowymi w następujący sposób:

```
def function_name(input_parameter):
    <ciało funkcji korzystające z input_argument>
    return output
```

### Print a Output
Return a Wyświetlanie: Instrukcja return jest używana do zwracania wartości z funkcji, która może być później użyta, podczas gdy print służy do wyświetlania wartości w konsoli tylko dla programisty.
