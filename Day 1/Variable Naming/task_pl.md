Poznaj zasady nazywania zmiennych w Pythonie.

## Zasady

1. Upewnij się, że nazwy Twoich zmiennych są opisowe  
2. Nie używaj spacji między słowami  
3. Nie zaczynaj od liczb  
4. Nie używaj specjalnych słów, takich jak print lub input  
5. Wybieraj proste słowa, które są mniej podatne na literówki  
6. Sprawdź wytyczne dotyczące stylu firmy, jeśli zaczynasz pracę w firmie
